In order to run the 6th coursework: (panda2cwk6) 

navigate to the folder in terminal or command prompt:

1. Type: node server/server_service.js and hit enter.
2. Type: ant judge in a different window.
3. Type: ant players in a different window.

If you do not have chai or mocha installed then:

Navigate to the server folder <your folder>/scotlandyard/server
Use the command npm install mocha
Use the command npm install chai
